<task type="auto">
  <name>Task 1: Create checkpoint manager module with fingerprinting</name>
  <files>src/checkpoint_manager.py, data/checkpoints/.gitkeep</files>
  <action>
Create `src/checkpoint_manager.py` with the following functions:

**`compute_dataset_fingerprint(df: pd.DataFrame) -> str`:**
- Sort DataFrame by all columns to ensure stable ordering
- Serialize to CSV string (without index)
- Compute SHA-256 hash of the serialized string
- Return hex digest as fingerprint
- Use pandas `.sort_values()` with all columns, then `.to_csv(index=False)`

**`save_checkpoint(checkpoint_dir: Path, config: dict, metadata: dict, model: BERTopic, embeddings: np.ndarray, dataset_fingerprint: str) -> None`:**
- Create checkpoint directory: `data/checkpoints/{timestamp}_{dataset_name}/`
  - timestamp format: `YYYYMMDD_HHMMSS` using `datetime.now().strftime('%Y%m%d_%H%M%S')`
  - dataset_name from metadata['dataset_name'], sanitized (replace spaces with underscores)
- Save config as JSON: `config.json` (UMAP, HDBSCAN, vectorizer, embedding model params)
- Save metadata as JSON: `metadata.json` (timestamp, dataset_name, topic_count, outlier_percentage, row_count)
- Save model using pickle protocol 5: `model.pkl` (use `pickle.dump(model, f, protocol=5)`)
- Save embeddings: `embeddings.npy` (use `np.save()`)
- Save dataset fingerprint: `dataset_fingerprint.txt` (plain text)
- Return the created checkpoint directory path

**`list_checkpoints() -> list[dict]`:**
- Scan `data/checkpoints/` for subdirectories
- For each directory, load `metadata.json` and return list of dicts with: path, timestamp, dataset_name, topic_count, outlier_percentage
- Sort by timestamp descending (newest first)
- Handle missing metadata.json gracefully (skip or mark as corrupted)

**`validate_checkpoint_integrity(checkpoint_dir: Path) -> tuple[bool, list[str]]`:**
- Check for required files: config.json, metadata.json, model.pkl, embeddings.npy, dataset_fingerprint.txt
- Check file sizes: model.pkl should be >1KB, embeddings.npy should be >100 bytes
- Try loading pickle with `pickle.load(f)` in try/except to detect corruption
- Return (is_valid, list_of_errors)

**`load_checkpoint(checkpoint_dir: Path) -> tuple[dict, dict, BERTopic, np.ndarray, str]`:**
- Validate integrity first using `validate_checkpoint_integrity()`
- If invalid, raise ValueError with error details
- Load and return: (config, metadata, model, embeddings, dataset_fingerprint)
